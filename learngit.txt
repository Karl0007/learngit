创建仓库
git init

更新
放到暂存区
git add <filename>
提交修改
git commit -m "<update message>"

查看更新信息
git log
git reflog

移动到节点
git reset --hard <commit id>

查看状态
git status

撤销当前工作区修改
git checkout -- <filename>

撤销当前暂存区修改
git reset HEAD <filename>

远程仓库
git remote add origin git@github.com:karl0007/<project name>

首次推送
git push -u origin master
